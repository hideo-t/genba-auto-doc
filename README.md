# genba-auto-doc

中堅ゼネコン向けの現場写真自動分類・完成図書生成システムのデモ。スマホで現場写真をアップロードし、AI（模擬）による工程別自動分類から完成図書PDFの生成までを体験できる営業プレゼン用Webアプリ。福島・仙台・いわき・郡山・会津の5つのサンプルプロジェクトを用意し、着工前・基礎工事・鉄骨建方・内装工事・完成の5段階で写真を分類。削減時間の定量表示により現場作業効率化を訴求。

## 技術スタック
- HTML5 / CSS3 / Vanilla JS
- Mobile-first responsive design
- Hosted on GitHub Pages
